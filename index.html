MANTENIMIENTO

El mantenimiento haciendo configuracón de redes virtuales VPN:Para configurar una red en activar la comunicación entre la red local o un recurso informático y el sistema se, debe configurar las subredes de la VCN con reglas de seguridad, esas reglas permiten el tráfico desde puertos y direcciones IP específicos, o rangos de puertos y direcciones IP, entre recursos; Esto basado en el modelo de referencia de interconexión de sistemas abiertos (OSI, Open System Interconection) lanzado en 1984 fue el modelo de red descriptivo creado por ISO. Para enfrentar el problema de incompatibilidad de redes, la organización internacional para la estandarización (ISO) investigó modelos de networking como la red de digital equipment corporation (DECnet), la arquitectura de sistemas de red (SNA) y el protocolo TCP/IP en el año 1970 a fin de encontrar un conjunto de reglas aplicables de forma general a todas las redes, con base en esta investigación en el aspecto hitórico la ISO desarrolló un modelo de red que ayuda a los fabricantes a crear redes que sean compatibles con otras redes. Algo completamente relacionado en los distintos aparatos que tengan su sistema operativo de cualquier índole.

INSTALACIÓN

Dejando claro que este taller únicamente lo hago en la ciudad de nuevo laredo no afuera, ofresco de forma extra instalaciones de software libre y/o programas anticensura y antiespionaje en concreto: a) FREENET e INTERNET INVISIBLE (i2p): software para poder conectarse a las redes más relevantes y dignas (desde mi punto de vista) siguen existiendo en internet. b)RETROSHARE: Un software que proporciona comunicaciones de red cifradas P2P(peer to peer), un sistema de correo electrónico no centralizado, mensajería instantánea, un BBS y un sistema para compartir ficheros basado en una red amigo-a-amigo, haciendo uso para todo ello de la herramienta de cifrado GPG para cualquier windows o para otros sistemas es una alternativa efectiva con la que crear tu propia red privada de comunicación. Aquí no se utilizan servidores centrales para interconectar a los usuarios. c) PSIPHON: Software para anonimizar las comunicaciones en windows y en android a través de una red privada virtual (VPN). d) LibreVPN: Es una red de malla virtual con secuencias de comandos de configuración que le permiten crear su propia VPN de malla. e) BitMessage: Es un protocolo de comunicación P2P utilizado para intercambiar mensajes cifrados de un usuario a muchos suscriptores. f) Anonet: Es una red descentralizada entre amigos que usa VPN y enrutadores BGP de software, se implementa como un grupo de túneles VPN (en su mayoría de igual a igual) a través de Internet y (generalmente basados ​​en Linux) enrutadores IP de habla BGP (principalmente IPv4). g) Red eDonkey: Es una red de intercambio de archivos punto a punto descentralizada, principalmente basada en servidor, es más adecuado para compartir archivos grandes entre usuarios y para proporcionar disponibilidad de archivos a largo plazo. Como la mayoría de las redes de intercambio, está descentralizado, ya que no existe un eje central para la red; Además, los archivos no se almacenan en un servidor central, sino que se intercambian directamente entre usuarios según el principio de igual a igual.
